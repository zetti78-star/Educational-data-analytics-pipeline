This project automates the extraction, transformation and visualization of educational data from Moodle.

It was designed to reduce manual work, eliminate errors and provide fast, reliable insights for both management and operational teams.

⚙️ Architecture

Extraction: SQL query (Moodle Configurable Reports)

Transformation: Power Query (Excel)

Visualization: Power BI + Excel reports

🔄 Data Flow

Moodle → SQL → CSV → Power Query →
➡ Power BI (management dashboard)
➡ Excel (detailed operational report)

📊 Outputs
🔹 Management Dashboard

Active vs Evaded students

Approved vs Failed

Filters by school/unit

🔹 Operational Report

Student-level data

Courses and activities

Engagement metrics

🚀 Impact

Reduced report generation time from hours to minutes

Eliminated manual processing errors

Automated reporting workflow

Improved decision-making

🛠️ Tools

SQL

Power Query

Power BI

Excel

📌 Notes

This project was developed based on a real-world educational scenario.
